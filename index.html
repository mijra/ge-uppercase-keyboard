Virtual keyboard for Georgian letters

This is demonstrational page to show you how can be Uppercase letters set to keyboard, and we show example, about how to use it in your document.

Unfortunately, copying that text will not show right letters for UPPERCASE, because it's not supported yet. That's why we created this demontrational page to ask unicode to support us.

Hint: use caps lock to see uppercase letters

Hint 2: you can edit text as well as use keyboard to press any key (Helper keys enabled - caps lock, alt, shift etc.).
